# samanthabaer.com

Website for Samantha Baer — professional event rider, host of The Elevated Equestrian podcast, and equestrian educator.

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **Auth:** Supabase (planned)
- **Payments:** Stripe (planned)
- **Hosting:** Vercel (recommended)

## Features

### Public Pages
- Home
- About
- Podcast (episode archive)
- Courses (purchase & subscription)
- Blog (SEO content)
- Contact
- Sponsors (partnership info)

### Member Features (Planned)
- User accounts
- Course access (individual purchase or subscription)
- Course progress tracking
- Member dashboard

### Admin Features (Planned)
- Upload/manage courses
- Manage blog posts
- View subscribers
- Track sales

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
